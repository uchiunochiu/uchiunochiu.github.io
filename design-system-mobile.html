📱 Mobile Design System（gluestack OSS）

React Native / Expo 向け。gluestack UI + NativeWind を前提に、寸法・状態・余白を固定化。
Source: gluestack OSS only No Pro assets Mobile-first

1) Foundations

TokenValueRule
Spacing4/8/12/16/20/24/32画面padding=16、カード間=12、内部gap=8
Radius8/10/12/16Input=10, Button/Card=12, Sheet=16
Type12/14/16/20本文14、見出し16〜20、補助12
ColorPrimary/Neutral/Success/Danger色だけで意味を伝えない（文言併記）
Tap targetmin 44x44icon-only操作も44以上必須

2) State Matrix

ComponentStatesRequired
Buttondefault / pressed / disabled / loading全variant実装
Inputdefault / focus / error / disablederror文言必須
Carddefault / selected / disabledselected時はborder強調
List itemdefault / pressed / destructive押下フィードバック120ms

3) Component Catalog（実装対象）

CategoryComponents (gluestack)規約
LayoutBox, VStack, HStack, Center, Grid, Dividerネスト3階層まで、gapトークン厳守
FormFormControl, Input, Textarea, Select, Checkbox, Radio, Switchラベル/説明/エラーを常設
ActionButton, Pressable, IconButton, FabPrimary/Secondary/Ghost/Destructive
FeedbackToast, Alert, Progress, Spinner, Skeleton, Badge非同期は必ずloading表示
OverlayModal, ActionSheet, Drawer, Popover, Tooltip閉じる導線を常に提供
DataCard, Table, Avatar, ImageEmpty/Error/Loadingの3状態を用意
