🖥️ PC Design System（gluestack OSS）

Web / Desktop 向け。gluestack UI + Tailwind で、管理画面の密度・可読性・状態遷移を統一。
Source: gluestack OSS only No Pro assets Desktop-first

1) Foundations

TokenValueRule
Grid12 columnmain 24px padding / widget gap 14px
Spacing4/8/12/16/20/24/32テーブル行8、セクション14
Radius8/10/12Input=8, Button=8, Card=12
Type12/13/14/16/20/24データ表は13、見出し16〜24
Motion120/180/240mshover/focus/transitionsで統一

2) State Matrix

ComponentStatesRequired
Buttondefault/hover/active/disabled/loadingPrimary, Secondary, Ghost, Danger
Input/Selectdefault/focus/error/disabledhelp + error文言
Table rowdefault/hover/selected選択時は背景差分
Carddefault/interactive/selected選択時border + shadow

3) Component Catalog（実装対象）

CategoryComponents (gluestack)規約
ShellBox, HStack, VStack, Grid, Dividerヘッダー・サイドバー・メインを固定テンプレ化
Data EntryFormControl, Input, Textarea, Select, Checkbox, Radio, Switch全フォームはラベル＋説明＋エラー
Data DisplayCard, Table, Badge, Avatar, Progress, SkeletonLoading/Empty/Errorを必須実装
OverlayModal, Drawer, Popover, Tooltip, Toast, AlertDialog確認操作はAlertDialog標準化
ActionButton, Pressable, Link, Menu主要CTAは1画面1つまで

4) Screen Recipes

ScreenTemplate

ListFilter bar + table + pagination
DetailOverview card + activity timeline + side actions
